# Server_Client
Sistema em Java fazendo conexão servidor-cliente por meio de sockets
- Executando primeiro o Server e em seguida o client, podemos estabelecer uma conexão.
- Objetivo principal do programa é fazer clientes produtores(Client) produzirem conteudo e armazenar os mesmos
no servidor(Server) e o cliente consumidor(Client) vai consumir esses conteudos.
